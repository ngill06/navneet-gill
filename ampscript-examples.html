AMPscript Code Examples

Here are some examples of AMPscript used in email templates. AMPscript is used for personalization and dynamic content in emails.

Example 1: Personalization


%%[
    VAR @FirstName, @SubscriberKey
    SET @SubscriberKey = _subscriberkey
    SET @FirstName = AttributeValue("FirstName")
]%%


Hi %%=v(@FirstName)=%%,


Welcome to our newsletter!

Example 2: Conditional Content


%%[
    VAR @OrderAmount
    SET @OrderAmount = AttributeValue("OrderAmount")
]%%

%%[ IF @OrderAmount > 100 THEN ]%%
    
Congratulations! You qualify for a special discount.

%%[ ELSE ]%%
    
Thank you for your purchase. Check out our new arrivals!

%%[ ENDIF ]%%
